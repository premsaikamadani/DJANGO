ORM:
> ORM stands for Object-Relational Mapping.
> It is a programming technique or layer that creates a bridge between object-oriented programming (00P)
languages and relational databases.
> ORM translates data between the objects in code and the database tables, enabling developers to work with database data using
objects without needing to write complex SQL queries directly.
